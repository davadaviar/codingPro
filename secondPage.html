Modul Pemrograman Front End Web Development

Penulis : Dava Daviar Saputra

Tujuan

Berikut beberapa tujuan dari penyusunan Modul Dasar Pemrograman (Front End) Web Development untuk anda :

  • Siswa diharapkan dapat mengetahui dasar pembuatan website terutama pada bagian (Front End).

  • Siswa dapat mengetahui dan memahami cara kerja website pada local komputer/laptop dan pada alat peramban (Browser seperti : Mozilla Firefox, Opera, Google Chrome, dll).

  • Siswa dapat menguasai penggunaan Markup Language seperti Hypertext Markup Language (HTML5), Cascading Style Sheet (CSS3) dan bahasa pemrograman Javascript untuk membuat sebuah website yang tepat guna.

  • Siswa dapat menggunakan Framework Bootstrap untuk membuat website dengan tampilan yang ramah pengguna dan tampilan yang modern.

  • Siswa dapat menggunakan repository berbasis cloud Github untuk menyimpan source code dan melakukan deployment ketika website sudah tampil secara langsung di internet (live) secara gratis.

  • Siswa dapat menciptakan sebuah protofolio produk berupa website

Persyaratan Tools

Berikut beberapa persyaratan tools yang digunakan selama belajar dan mempraktekkan materi yang ada pada modul ini :

  • Komputer/laptop minimum processor Intel core i3 atau versi ke atas.

  • Text editor seperti Visual Studio Code, Sublime, Atom, Vim sesuai dengan selera siswa.

  • Komputer/laptop telah terinstall dengan Browser seperti Chrome, Mozilla Firefox, Microsoft Edge dll.

  • Terminal yang sudah tersedia pada laptop / komputer seperti Microsoft Power Shell, Git Bash, atau pun Command Prompt.

Mekanisme Pembelajaran

Untuk mendapatkan hasil yang maksimal dengan menggunakan modul ini siswa dapat melakukan mekanisme pembelajaran seperti berikut :

  • Disarankan menggunakan komputer/laptop untuk pengalaman belajar yang lebih optimal karena materi yang ada pada modul ini bersifat praktis yang dapat langsung dilakukan.

  • Bentuk materi yang terdapat pada modul ini dominan berupa teks bacaan dengan konten yang berisi project based learning.

  • Untuk mengevaluasi kompetensi siswa yang telah mempelajari modul ini adalah dengan membuat sebuah tugas akhir dengan mengimplementasikan semua materi yang telah diajarkan. Jika siswa diketahui membuat tugas akhir dengan source code di luar materi ini siswa dianggap plagiasi dan terhitung gagal di kelas ini.

  • Siswa dapat bertanya dan berdiskusi jika mengalami kendala pada penjelasan modul di forum diskusi yang telah disediakan di setiap submodul.
