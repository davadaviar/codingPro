Pengertian Server & Client

Kita sering sekali melihat tampilan website yang bagus ketika menggunakan browser. Tampilan web pada browser sudah tertata rapi dan mudah kita mengerti namun dibalik itu browser menerjemahkan HTML, CSS dan Javascript yang disimpan pada server. Untuk mempermudah pemahaman terhadap struktur website bisa digambarkan dengan struktur tubuh manusia beserta fungsinya.

HTML berfungsi untuk pembentuk bagain dari website seperti bagian header, navbar, button dll. Kemudian CSS sebagai pengatur tata letak bagian website serta menjadi pemberi warna pada website dan yang tak kalah penting adalah Javascript yang berfungsi untuk interaksi dari suatu website ketika kita mengeklik suatu bagian website muncul seperti pop up, berganti warna, sehingga website tersebut berfungsi sesuai dengan kebutuhan penggunanya atau bisa disebut website yang dinamis.
