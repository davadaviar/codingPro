Pengertian Server & Client

Web Server

Banyak diantara kita yang mengira bahwa server merupakan sebuah komputer canggih yang memiliki performa tinggi dan berukuran besar. Namun hal tersebut tidaklah salah, memang komputer yang berfungsi sebagai server haruslah memiliki performa yang tinggi dikarenakan server memiliki tugas yang cukup berat. Fungsi server yaitu sebagai penyedia, pengelola, danpenyimpan data yang dibutuhkan oleh client. Server tidak hanya melayani satu client dalam satu waktu, namun bisa beribu-ribu bahkan berjuta-juta client dalam satu waktu. Di sinilah HTML, CSS, dan Javascript disimpan.

HTTP Server

Merupakan sebuah perangkat lunak (software) yang berfungsi untuk menerima transaksi hypertext protocol yang terdapat di dalam website. Banyak sekali software dari HTTP server ini seperti contohnya Apache yang gampang digunakan, gratis dan tersedia pada seluruh komputer berbasis UNIX seperti (MacOs) dan Windows.

DNS Server

Setiap komputer, laptop, smartphone atau pun router yang terhubung ke internet semuanya memiliki sebuah IP Address. Seperti Google Sendiri yang memiliki IP 8.8.8.8. Jika kita mengakses IP Address tersebut maka akan muncul halaman Google. Namun apakah kita harus menghafalkan angka - angka tersebut untuk mengakses halaman Google? Jika satu halaman saja maka mudah menghafalnya. Namun jika banyak halaman yang kita akses sungguh tidak efektif untuk menghafalkan banyak kombinasi angka. Dengan demikian maka DNS (Domain Name System) inilah yang menerjemahkan IP Address 8.8.8.8 menjadi google.com yang mudah untuk diingat.

Client

Jika ada penyedia (server) pasti ada pengguna (client). Client adalah sebuah perangkat yang melakukan permintaan (request) layanan tertentu kepada server. Di sini client berupa browser yang kita gunakan. Browser akan melakukan transaksi request melalui HTTP dalam melakukan transaksi datanya. Dengan demikian borwser dapat menyajikan sebuah informasi dengan tampilan yang indah dan mudah dimengerti oleh pengguna seperti kita ketika mengakses sebuah halaman web dari data mentah yang disimpan pada web server.

Server-side & Client-side

Dalam pengembangan web, dua istilah mengenai server-side dan client-side sudah sangat umum. Client-side adalah sebuah proses olah data di sisi client, client akan meminta data dari server kemudian akan diolah di sisi client. Biasanya pengolahan data pada sisi client ini berupa HTML, CSS dan Javascript.Sedangkan server side, lebih merujuk ke sisi server yang dimana pada proses ini terjadinya interaksi antar database yang dimana client tidak harus mengetahui proses tersebut dan memang tidak boleh tahu untuk pengolahan data yang ada di dalam server tersebut sehingga client hanya diberikan hasil olahan dari pengolahan data yang terjadi di server.
