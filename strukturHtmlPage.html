Struktur HTML

Langsung saja kita membahas markup language pembentuk sebuah web yaitu HTML. HTML memiliki dua bagian penting yaitu Head dan Body. Untuk lebih jelasnya bisa melihat bagian berikut ini.

Berikut adalah onlinde code editor yang bisa digunakan untuk mempraktikkan materi pada modul ini. Cara penggunaannya sebagai berikut, klik menu bertuliskan Show files di pojok kiri atas, lalu klik file index.html
