Struktur HTML

Langsung saja kita membahas markup language pembentuk sebuah web yaitu HTML. HTML memiliki dua bagian penting yaitu Head dan Body. Untuk lebih jelasnya bisa melihat bagian berikut ini.

Pada code di atas adalah sebuah sintaks yang menunjukkan bahwa pada file ini memiliki unsur elemen HTML. Sintak tersebut berada pada awal bagian ketika kita akan menuliskan kode HTML dan diakhiri dengan tag html seperti kode di atas.

Untuk bagian header secara umum berisikan tag-tag yang telah disebutkan di atas. Kemudian untuk pembahasan selanjutnya adalah bagian Body. Di bagian ini lah kita dapat menuliskan kode HTML kita. Dan di dalam tag body ini juga kita dapat meletakkan sebuah file script yang akan digunakan di website yang kita buat. Dan elemen yang ada di dalam tag body inilah yang nanti akan dirender/ditampilkan pada web browser kita menjadi sebuah halaman yang mudah kita pahami.

Hal yang penting kita pahami adalah HTML dan CSS bukanlah bahasa pemrograman. Banyak di antara kita yang menganggap HTML dan CSS adalah sebuah bahasa pemrograman. HTML dan CSS hanyalah sebuah markup language yang di mana mereka hanya sebagai pembangun struktur dari sebuah website. Dan selain itu HTML dan CSS di dalam penulisannya tidak terkandung logika sama sekali sehingga tidak dimasukkan ke dalam kategori bahasa pemrograman. Sedangkan Javascript adalah sebuah bahasa pemrograman karena dengan Javascript kita dapat membuat fungsi-fungsi dan meletakkan sebuah logika /algoritma yang membuat sebuah website menjadi website yang fungsional dan interaktif bagi pengguna yang membutuhkan.
