Struktur HTML

Langsung saja kita membahas markup language pembentuk sebuah web yaitu HTML. HTML memiliki dua bagian penting yaitu Head dan Body. Untuk lebih jelasnya bisa melihat bagian berikut ini.

Berikut adalah code editor yang bisa digunakan untuk mempraktikkan materi pada modul ini.

Pada project di atas bisa kita amati bahwa pada sebuah elemen HTML terdapat bagian Head yang berisikan beberapa tag HTML seperti meta data, title, dan link yang digunakan untuk mensinkronkan file CSS kita pada HTML tersebut.
