Tugas Akhir

Silabus

Kalian telah menyelesaikan seluruh materi yang ada pada modul ini. Sangat tidak afdhol jika kita tidka mengimplementasikan ilmu - ilmu yang telah kita pelajari di modul ini. Sebagai bentuk implementasi dari modul ini siswa diharap membuat sebuah project sederhana dengan memanfaatkan seluruh dasar - dasar yang telah dijelaskan pada modul ini dari awal hingga akhir. Berikut petunjuk, syarat dan ketentuan untuk mengerjakan tugas akhir ini :

Untuk mengerjakan tugas akhir berupa proyek siswa diminta untuk menggunakan kode editor berupa Visual Studio Code, Atom, Sublime, dan lain - lain. Dan file dari project tugas akhir dapat anda submit pada halaman ini.

Untuk ketentuan pengerjaan project tugas akhir beserta nilai yang akan siswa dapatkan berdasarkan implementasi project pada tugas akhir yang didasarkan pada materi modul ini sebagai berikut:

  • Siswa akan mendapat nilai 100 jika menerapkan Semantic HTML, CSS grid, DOM Javascript, Fetching API, dan menggunakan Framework Bootstrap dalam projectnya.

  • Siswa akan mendapat nilai 90 jika menerapkan Semantic HTML, CSS grid, DOM Javascript dan Fetching API dalam projectnya.

  • Siswa akan mendapat nilai 80 jika menerapkan Semantic HTML, CSS grid, DOM Javascript, dan menggunakan Framework Bootstrap dalam projectnya.

  • Siswa akan mendapat nilai 70 jika menerapkan Semantic HTML, CSS grid, dan DOM Javascript dalam projectnya.

Syarat - syarat yang harus siswa ikuti untuk penilaian tugas akhir sebagai berikut :

  • Yang harus ada pada file adalah file index.html, folder src, dan file README.md yang berfungsi untuk memberi definisi project anda.

  • Jika terdapat plagiarisme pada project anda dengan project siswa lain, maka nilai dari siswa yang anda plagiat dan project anda otomatis diberi nilai 0

  • Siswa dapat mengirimkan link repository Github atau file berformat .rar/.zip dari project akhir.
