Tag - Tag HTML

Struktur HTML sudah dijelaskan pada modul sebeblumnya. Pada kali ini kita akan mempelajari tag - tag HTML yang sering digunakan seorang web developer ketika membuat sebuah website. Untuk mengetahui lebih lengkapnya, yuk langsung saja kita lanjutkan penjelasaannya di bawah ini.

Tag Heading

Tag heading merupakan tag yang digunakan untuk menampilkan elemen heading pada sebuah website. Tag ini biasanya digunakan untuk bagian judul dari sebuah konten yang ada di website. Berikut macam - macam dari tag heading pada HTML.
