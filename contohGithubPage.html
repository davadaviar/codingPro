Contoh Repository Github

Silabus

Jika kita telah membuat sebuah website di lokal laptop atau komputer kita, kita hanya bisa mengakses di komputer kita dan hanya kita saja yang mengetahuinya. Untuk membuat website dapat dilihat oleh orang lain melalui internet dan orang lain tidak hanya bisa melihat halaman website kita saja serta orang lain juga bisa mengetahui susunan kode website kita melalui repository website yang telah kita unggah menggunakan Github.

Selain itu Github memiliki fungsi yang penting bagi developer yaitu sebagai wadah/repository portofolio project yang mereka miliki. Bagi seorang calon developer website yang ingin mendapatkan pekerjaan di perusahaan portofolio adalah senjata utama untuk mendapatkan kesempatan tersebut. Maka dari itu menata rapi profil Github agar menarik sebagai portofolio adalah salah satu langkah penting yang bisa kita lakukan.
