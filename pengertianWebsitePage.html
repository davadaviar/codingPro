Apakah Website itu?

Website merupakan sebuah halaman yang memuat informasi berupa teks, gambar, atau pun video. Website dapat diakses menggunakan browser yang terhubung dengan internet. Website dapat saling terhubung setiap bagiannya dengan menggunakan hypertext link. Perkembangan website sekarang termasuk sangat cepat karena di awal kemunculannya website hanya digunakan untuk sebuah organisasi untuk mengaksesnya namun di masa sekarang perorangan sudah bisa mengaksesnya. Pada 30 tahun silam, Tim Berners - Lee menemukan World Wide Web (WWW) untuk pertama kalinya untuk mengakses sebuah website yang dipergunakan untuk saling berbagi informasi terhadap sesama ilmuwan dan universitas saja.

Contoh Website

Berdasarkan fungsionalitasnya website dibagi menjadi dua yaitu website statis dan website dinamis. Untuk mengetahui lebih dalam berikut contoh beserta penjelasannya.

Dari ilustrasi di atas yang dimaksud website statis adalah website yang berfungsi untuk menampilkan informasi saja tanpa ada unsur pengguna melakukan sebuah aksi seperti mengirim pesan, memasukkan sebuah data pada form dll. Website statis ini biasanya digunakan sebagai halaman portofolio seseorang untuk menjelaskan beserta menawarkan keahlian profesionalnya kepada calon client.

Sedangkan website dinamis seperti contoh di atas, pengguna dapat melakukan sebuah proses yaitu membeli sebuah barang sampai proses selesai. Pengguna dapat melakukan sebuah aksi memilih barang, menambah dan mengurangi jumlah barang, melakukan pembayaran dll. Sehingga untuk membuat website yang dinamis dibutuhkan dua aspek yaitu front end development dan back end development agar fungsionalitas website sesuai dengan kebutuhan pengguna.
